# 📱 Apagado y Encendido de Celulares - Sistema de Protección Móvil

## 🎯 Descripción del Proyecto

**Apagado y Encendido de Celulares** es un programa diseñado para proteger la identidad y datos personales de personas mayores de 18 años que han perdido o les ha robado su teléfono móvil. 

El sistema permite apagar remotamente un dispositivo de manera segura, evitando que terceros accedan a información sensible como:
- Datos bancarios
- Información personal
- Cuentas de redes sociales
- Archivos y fotos privadas
- Documentos de identidad digitales

---

## ✨ Características Principales

✅ **Apagado Remoto**: Apaga tu dispositivo desde cualquier ubicación
✅ **Protección de Datos**: Evita acceso no autorizado a información sensible
✅ **Interfaz Amigable**: Diseñado para usuarios de todas las edades
✅ **Seguridad Verificada**: Requiere autenticación para evitar uso no autorizado
✅ **Soporte Múltiples Dispositivos**: Compatible con varios modelos de teléfonos
✅ **Respuesta Rápida**: Acción inmediata al activarse

## 🔒 Requisitos y Compatibilidad

### Requisitos Mínimos:
- **Edad**: Mayor de 18 años
- **Sistema Operativo**: Android 5.0+ o iOS 10+
- **Conexión a Internet**: Requerida para funcionamiento remoto
- **Espacio Disponible**: Mínimo 50 MB en dispositivo
- **Permisos**: Acceso a administrador del dispositivo

### Dispositivos Compatibles:
- Teléfonos inteligentes Android (5.0 y superior)
- Dispositivos iOS (10.0 y superior)
- Tabletas compatibles

## 🚀 Guía de Instalación

### Paso 1: Descargar la Aplicación
1. Abre Google Play Store (Android) o App Store (iOS)
2. Busca "Apagado y Encendido de Celulares"
3. Haz clic en "Instalar"

### Paso 2: Configuración Inicial
1. Abre la aplicación
2. Acepta los términos y condiciones
3. Crea una cuenta con:
   - Correo electrónico válido
   - Contraseña segura (mín. 8 caracteres)
   - Número de teléfono de recuperación

### Paso 3: Activar Permisos Necesarios
1. Ve a **Configuración**
2. Selecciona **Administradores de Dispositivo**
3. Activa los permisos para la aplicación

### Paso 4: Verificación de Seguridad
1. Responde las preguntas de seguridad
2. Verifica tu correo electrónico
3. ¡Listo! Tu dispositivo está protegido

## 📖 Cómo Usar la Aplicación

## 🆘 Preguntas Frecuentes (FAQ)

### ¿Qué pasa si apago mi teléfono por accidente?
Puedes encenderlo manualmente presionando el botón de encendido. Si necesitas ayuda, contacta a nuestro equipo de soporte.

### ¿Cómo puedo recuperar mi dispositivo después de apagarlo?
La aplicación no recupera dispositivos. Para recuperar tu teléfono robado, debes:
- Rastrearlo mediante Google Find My Mobile o iCloud
- Contactar a la policía local
- Usar servicios de rastreo de terceros

### ¿Mi contraseña es segura?
Usamos encriptación de nivel militar (AES-256). Nunca compartas tu contraseña con nadie.

### ¿Funciona sin conexión a Internet?
No, el apagado remoto requiere conexión a Internet. El apagado local no lo requiere.

### ¿Cuánto cuesta?
Versión básica: **Gratuita**
Versión Premium: **$4.99 USD/mes** (con soporte prioritario)

### ¿Es legal usar esta aplicación?
Sí, es completamente legal si es para proteger tu propio dispositivo. El uso para apagar dispositivos de terceros es ilegal.

### ¿Qué información personal recopilan?
Solo recopilamos:
- Nombre y correo electrónico
- Número de teléfono
- Información del dispositivo (modelo, IMEI)

Consulta nuestra [Política de Privacidad](privacy-policy.md) para más detalles.

## 🔧 Solución de Problemas

### La aplicación no se abre
- Reinicia tu dispositivo
- Verifica que tengas espacio suficiente
- Desinstala y reinstala la aplicación

### No puedo apagar remotamente
- Verifica tu conexión a Internet
- Asegúrate de que la aplicación tiene permisos de administrador
- Comprueba que tu sesión está activa

### Olvidé mi contraseña
1. Ve a la pantalla de inicio de sesión
2. Haz clic en "¿Olvidaste tu contraseña?"
3. Ingresa tu correo electrónico
4. Verifica el enlace enviado a tu bandeja de entrada
5. Crea una nueva contraseña

### No recibo correos de verificación
- Revisa tu carpeta de spam
- Añade nuestro correo a contactos
- Solicita reenvío desde la aplicación
- Contacta a soporte técnico
